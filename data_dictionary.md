# Mutual Fund Data Dictionary

## amfi_code
Type: TEXT
Description: Unique AMFI mutual fund scheme identifier.

## scheme_name
Type: TEXT
Description: Name of the mutual fund scheme.

## fund_house
Type: TEXT
Description: Asset Management Company (AMC) managing the fund.

## category
Type: TEXT
Description: Fund category.

## nav
Type: REAL
Description: Net Asset Value of the mutual fund.

## date
Type: DATE
Description: NAV or transaction date.

## transaction_type
Type: TEXT
Description: SIP, Lumpsum, or Redemption.

## amount_inr
Type: REAL
Description: Transaction amount in Indian Rupees.

## expense_ratio_pct
Type: REAL
Description: Annual expense ratio charged by the fund.

## return_1yr_pct
Type: REAL
Description: One-year return percentage.

## return_3yr_pct
Type: REAL
Description: Three-year return percentage.

## return_5yr_pct
Type: REAL
Description: Five-year return percentage.
